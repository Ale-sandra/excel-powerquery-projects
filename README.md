## 📊 Excel Data Analysis Exercises

This repository contains Excel exercises developed during a data analyst training program.

The work focuses on practical data analysis tasks, including data cleaning, transformation and reporting.

### 🛠 Skills Applied
- Pivot Tables & Charts  
- Power Query  
- Data cleaning  
- KPI analysis  

### 🎯 Purpose
These exercises were designed to build practical skills in analyzing datasets and producing clear, structured reports.
